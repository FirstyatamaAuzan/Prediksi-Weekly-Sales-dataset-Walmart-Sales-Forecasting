# Prediksi-Weekly-Sales-dataset-Walmart-Sales-Forecasting
Proyek Prediksi Penjualan Walmart menggunakan model regresi untuk meramalkan penjualan mingguan produk di berbagai toko Walmart berdasarkan dataset historis yang mencakup fitur seperti harga  markdown dan lain lain.

**Deskripsi Proyek**

Proyek ini bertujuan untuk memprediksi Weekly Sales dari data penjualan Walmart menggunakan beberapa model regresi seperti XGBoost, Random Forest Regressor (RF), dan Gradient Boosting Regressor (GBT). Dataset yang digunakan meliputi data historis penjualan toko Walmart, data fitur cuaca, harga bahan bakar, dan informasi lainnya terkait dengan masing-masing toko.

**# Alat**
Python 3.x
Apache Spark (untuk memproses data besar menggunakan PySpark)
Pandas (untuk manipulasi data)
Matplotlib dan Seaborn (untuk visualisasi)
XGBoost (untuk pemodelan regresi)
scikit-learn (untuk evaluasi dan metrik)

**Instalasi**
Instalasi Apache Spark dan PySpark:
      "pip install pyspark"
Instalasi XGBoost:
      "pip install xgboost"
Instalasi dependensi lainnya:
      "pip install pandas matplotlib seaborn numpy scipy"
